# Umrechner-US-Dollar-in-Euro

Angabe:

Umrechnung von US-Dollar in Euro (18 Punkte)
Schreiben Sie ein Programm zur Umrechnung von US-Dollar in Euro.
Das Programm soll zunächst die Anzahl von US-Dollar-Banknoten und -Münzen (jeweils die Anzahl von 100-Dollar-, 50-Dollar-, 20-Dollar-, 10-Dollar-, 5-Dollar-, 2-Dollar- und 1-Dollar-Scheinen sowie 1-Dollar-, 50-Cent-, 25-Cent-, 10-Cent-, 5-Cent- und 1-Cent-Münzen) sowie den aktuellen Umrechnungskurs einlesen, und diese in Euro-Bargeld, d.h. in die benötigte Anzahl von Euro-Banknoten und -Münzen, umrechnen.
Verwenden Sie für jede Euro-Banknote bzw. -Münze eine eigene Variable. Beachten Sie, dass der Gesamtbetrag in US-Dollar in größtmögliche Euro-Scheine und -Münzen umgerechnet werden soll. Geben Sie anschließend den Gesamtbetrag in US-Dollar und Euro sowie die Anzahl der Euro-Scheine und -Münzen aus.
